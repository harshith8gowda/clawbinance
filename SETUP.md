# ClawBinance — Quick Start Guide

## 🚀 Installation

### Step 1: Clone Repository
```bash
git clone https://github.com/yourusername/clawbinance.git
cd clawbinance
```

### Step 2: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 3: Configure Environment
```bash
cp .env.example .env
# Edit .env with your API keys
```

### Step 4: Run ClawTrader (Option A)
```bash
python ClawTrader/binance_trader.py
```

### Step 5: Run ClawBrief (Option B)
```bash
python ClawBrief/daily_brief_agent.py
```

## 🔑 Getting API Keys

### Binance API
1. Log in to https://www.binance.com
2. Go to Account → API Management
3. Create new API key
4. **Enable "Futures" but NOT "Withdrawals"**
5. Copy API Key and Secret to `.env`

### Telegram Bot
1. Message @BotFather on Telegram
2. Create new bot with `/newbot`
3. Copy token to `TELEGRAM_BOT_TOKEN`
4. Message your bot, then visit:
   `https://api.telegram.org/bot<token>/getUpdates`
5. Find your chat ID in the response

## 🛠️ Troubleshooting

### "Error fetching data"
- Check your Binance API keys
- Ensure you have internet connection
- Verify API key has Futures permissions enabled

### "Telegram error"
- Check bot token is correct
- Verify chat ID is correct (must be your user ID, not bot ID)
- Ensure you've messaged the bot first

### "No signals generated"
- This is normal! Markets are often in "hold" mode
- Z-Score only triggers on extreme deviations
- Check that markets are actually moving
